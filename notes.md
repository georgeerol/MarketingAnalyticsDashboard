This folder contains a model trace sample_mmm.pkl
PLEASE COPY THE FILE AND DO NOT REMOVE IT

This model trace is a google meridian model trace. The developer documentation is here: Module: meridian

It contains all required data and transformation functions.

Concerning the plots, you must create the following two charts:
	•	A contribution chart (pick one)
	•	The response curves

(ref: https://developers.google.com/meridian/docs/advanced-modeling/interpret-visualizations)

The general structure of the dashboard and other visualizations are up to you.
The goal is to create a compelling narrative for a customer, that shows the performance of their channels.
